# nmBot Terminology Glossary

Updated: 2026-04-09

## Scope

- This glossary standardizes nmBot terms across UI copy, localization, documentation, and product communication.
- Entries are aligned with existing locale values in en, zh-hans, and zh-hant.

## Ordering Method

- Primary key: `Domain` using a fixed domain order.
- Secondary key: `Source Term` in ascending alphabetical order (A-Z) within each domain.
- Domain order:
  1.  Governance and Permissions
  2.  Moderation and Safety
  3.  Messaging and Automation
  4.  AI and Interaction
  5.  Platform and Commerce

## Field Definitions

| Field Name            | Description                                                                                    |
| --------------------- | ---------------------------------------------------------------------------------------------- |
| Domain                | Functional domain used for deterministic ordering and easier lookup.                           |
| Source Term           | Canonical English term used as the baseline for product and feature naming.                    |
| Target Term (zh-Hans) | Simplified Chinese translation.                                                                |
| Target Term (zh-Hant) | Traditional Chinese translation.                                                               |
| Part of Speech        | Core grammatical category (for example n., v., v. phrase).                                     |
| Definition            | Project-specific meaning and scope in nmBot.                                                   |
| Context/Usage         | Typical usage in UI, settings pages, or product copy.                                          |
| Forbidden Terms       | Incorrect, ambiguous, or discouraged alternatives. Multilingual wrong examples are encouraged. |
